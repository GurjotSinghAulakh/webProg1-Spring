Oppgave 1 :

Løsning:

Metode 1:



Metode 2:

Oppgave 3:

Lag et input-felt i en HTML-side. Lag et JavaScript som leser dataene fra input-feltet når det er endringer i feltet (hendelsen "onChange") og som skriver disse dataene ut på en blank side med "document.write". Funksjonen som kalles ved "onChange" skal ta en inn-parameter, som er verdien av input-feltet.

Løsning:

Skriv melding:



HTML :

<p>Skriv melding: <label> <input type="text" onchange="skrivMelding(this.value);" /><br /> </label> </p>

JavaScript (JS) :

function skrivMelding(innData) { let utData = "Du har skrevet inn følgende data: " + innData + "."; document.write(utData); }

Oppgave 6:

Utvid oppgave 3 ved å gi et farge til teksten som vises på blank side.

Løsning:

<p>Melding får en blått farge: <label> <input type="text" onchange="fargetMelding(this.value);" /><br /> </label> </p>

Oppgave 9:

Løsning:

HTML :

<button onclick="tellOpp()">Tell antall click</button>

JavaScript (JS) :

let utData = 0; function tellOpp() { utData ++; console.log(utData); }


Oppgave 10:

Endre oppgave 9 ved å legge til en melding til console slik at den viser "Du har trykket på knappen X ganger" (hvor X er antall ganger knappen ble trykket på).

Løsning:

HTML :

<button onclick="tellOppMelding()">Meldding om antall click</button>

JavaScript (JS) :

let ut = 0; function tellOppMelding() { ut++; console.log("Du har trykket på knappen " + ut + " ganger."); }
